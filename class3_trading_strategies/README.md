# Class 3: Trading & Investing Strategies

**Duration**: 4 weeks (20 lessons)
**Time per lesson**: 45 minutes (30 min lecture + 15 min hands-on)
**Prerequisites**: Class 1 (Fundamentals) and Class 2 (Technical Analysis)

---

## Overview

Class 3 covers strategy frameworks from active day trading to long-term value investing. You'll learn multiple approaches to the markets and discover which styles match your personality, time commitment, and goals.

**Key Outcomes:**
- Understand different trading timeframes and styles
- Apply both technical and fundamental analysis to strategies
- Build systematic trading and investing approaches
- Learn options basics for income generation

Before starting Class 3, ensure you can:

- [ ] Place different order types (market, limit, stop)
- [ ] Explain the difference between trading and investing
