## Prerequisites

- Basic computer skills
- No prior financial knowledge required
- Python experience helpful but not required

## How to Use

### Google Colab (Recommended)
1. Click the "Open in Colab" badge in any notebook
2. Run the setup cell first
3. Follow along with the lesson

### Local Installation
```bash
cd money-talks
pip install -r requirements.txt
jupyter notebook class1_fundamentals/
```

## Time Commitment

- **Lecture**: 30 minutes per day
- **Hands-on**: 15 minutes per day
- **Total**: 45 minutes per day, 5 days per week
- **Class Duration**: 4 weeks (15 hours total)

## Key Skills You'll Gain

By the end of Class 1, you will be able to:
- Explain how financial markets work
- Identify different asset classes and their characteristics
- Choose appropriate order types for different scenarios
- Read candlestick charts and volume patterns
- Manage trading psychology and emotions
- Use Python to analyze stock data with yfinance
