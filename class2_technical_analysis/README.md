# Class 2: Technical Indicators & Analysis

Master chart analysis and indicator interpretation with 20 comprehensive lessons.

**Duration**: 4 weeks (20 lessons)
**Time per lesson**: 45 minutes (30 min lecture + 15 min hands-on)
**Prerequisites**: Class 1 - Trading & Investing Fundamentals

---

## Course Overview

Class 2 teaches you how to analyze price charts using technical indicators. You'll learn to identify trends, measure momentum, assess volatility, and find key price levels for trading decisions.

### What You'll Learn

- **Trend Indicators**: Moving averages, crossovers, and trend identification
- **Momentum Indicators**: RSI, MACD, Stochastic, and divergence analysis
- **Volatility & Volume**: Bollinger Bands, ATR, OBV, VWAP
- **Building Systems**: Support/Resistance, trendlines, Fibonacci, pivot points

## Technical Analysis Toolkit Summary

After completing Class 2, you'll have mastered:

| Category | Indicators | Primary Use |
|----------|------------|-------------|
| **Trend** | SMA, EMA, MA Crossovers | Direction & trend strength |
| **Momentum** | RSI, MACD, Stochastic | Overbought/oversold, divergence |
| **Volatility** | Bollinger Bands, ATR | Volatility measurement, stops |
| **Volume** | OBV, VWAP | Confirmation, institutional levels |
| **Price Levels** | S/R, Trendlines, Fib, Pivots | Entry/exit targets |
